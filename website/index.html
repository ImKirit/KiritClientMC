KiritClient

A sleek, lightweight Minecraft launcher with multi-account support, mod management, custom capes, and a modern glass UI.

Download for Windows View Source

Windows 10/11 · 4.5MB Installer · Free & Open Source

Multi-Account

Switch between multiple Microsoft accounts seamlessly with secure token storage.

Mod Loaders

Fabric, Forge, NeoForge, and Quilt support with built-in Modrinth mod search.

Profiles

Create unlimited profiles with different versions, loaders, mods, texture packs, and shaders.

Custom Capes

Upload your own cape visible to all KiritClient users in-game.

Customizable

Change background themes, accent colors, and toggle glass effects to make it yours.

Lightweight

Built with Tauri 2 — only ~6MB installer. No bloated Electron, no Chromium bundled.
